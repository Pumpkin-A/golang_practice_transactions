ЛР1 Знакомство со стандартным http
В данной лр требуется создать http-сервер для генерации случайных данных в формате json.
 Структура данных представляет собой описание данных платежа - uuid транзакции, 
 тип платежа (просто строка из заданного заранее списка вариантов), 
 дата транзакции (2023 год), сумма платежа (от 50 от 1000 долларов), 
 айди отправителя (число), айди получателя (0-100, например).

Требуемые обработчики сервера:
GET /generate/transactions - возвращает массив платежей (параметр - их количество)

Доп. задание: сделать POST обработчик для изменения стандартных параметров генерации
